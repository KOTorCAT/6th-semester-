# Отчёт о запуске контейнера

## Начальное состояние
- Чистая установленная ОС в облаке (WSL)

## Выполненные действия

1. Установлен Docker
2. Запущен контейнер на основе образа **nginx**:
   ```bash
   docker run -d -p 3000:80 --name my-webapp nginx
   ```

3. Проверка работы контейнера:
   ```bash
   docker ps
   ```
