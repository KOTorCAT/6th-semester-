# Лабораторная работа №1: Прогнозирование оттока клиентов

## Цель
Построить модель логистической регрессии для прогнозирования оттока клиентов банка.

---

## Установка и запуск

```bash
pip install -r requirements.txt
python churn_model.py
```

---

## Результаты

| Метрика | Значение |
|---------|----------|
| Accuracy | 93.9% |
| Precision | 78.4% |
| Recall | 96.0% |
| F1-score | 0.863 |
| ROC-AUC | 0.990 |

---

## Выходные файлы
- `confusion_matrix.png` — матрица ошибок  
- `roc_curve.png` — ROC-кривая  
- `feature_importance.png` — важность признаков  
- `metrics.png` — сравнение метрик  

---

## Ключевые выводы

- **Активность клиента** — главный фактор: активные клиенты уходят реже  
- **Возраст** — чем старше, тем выше риск оттока  
- **География** — в Германии отток повышен  
- Модель **выявляет 96%** уходящих клиентов (Recall), что позволяет оперативно реагировать
