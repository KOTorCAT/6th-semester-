## Цель и задачи

**Цель работы** — построить нейросетевую модель, способную по 13 физико-химическим параметрам определять сорт вина (один из трёх классов).

Для достижения цели решались следующие задачи:

- разделение выборки на обучающую, валидационную и тестовую части по фиксированной схеме;
- проектирование и обучение нейронной сети, обеспечивающей точность выше **94%** на тестовом наборе;
- фиксация архитектуры и числа обучаемых параметров модели.

## Реализация (код)

```python
import numpy as np
from sklearn.datasets import load_wine
from sklearn.model_selection import train_test_split
from tensorflow.keras import utils
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense
from tensorflow.keras.optimizers import Adam

# Загрузка и кодирование целевой переменной
data = load_wine()
X = data.data
y = utils.to_categorical(data.target, num_classes=3)

# Фиксированное разбиение данных
X_all, X_test, y_all, y_test = train_test_split(
    X, y, test_size=0.1, shuffle=True, random_state=6
)
X_train, X_val, y_train, y_val = train_test_split(
    X_all, y_all, test_size=0.1, shuffle=True, random_state=6
)

# Построение архитектуры
model = Sequential([
    Dense(128, input_dim=X_train.shape[1], activation='relu'),
    Dense(64, activation='relu'),
    Dense(3, activation='softmax')
])

# Компиляция и сводка по модели
model.compile(
    optimizer=Adam(learning_rate=0.001),
    loss='categorical_crossentropy',
    metrics=['accuracy']
)
model.summary()

# Цикл обучения
history = model.fit(
    X_train, y_train,
    epochs=100,
    batch_size=16,
    validation_data=(X_val, y_val),
    verbose=1
)

# Финальная оценка на тесте
test_loss, test_acc = model.evaluate(X_test, y_test)
print(f"\nТочность на тестовой выборке: {test_acc * 100:.2f}%")
```

|------------|----------------|-------------------|------------|
| Входной    | 13             | —                 | принимает признаки вина |
| Скрытый 1  | 128            | ReLU              | выделение нелинейных паттернов |
| Скрытый 2  | 64             | ReLU              | дальнейшее обобщение признаков |
| Выходной   | 3              | Softmax           | формирует вероятности принадлежности к трём классам |

**Оптимизатор** — Adam с learning rate = 0.001, функция потерь — categorical_crossentropy (подходит для многоклассовой классификации).
