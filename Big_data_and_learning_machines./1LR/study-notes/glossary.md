# Глоссарий

## Mutual Information
Мера взаимной информации между признаком и целевой переменной.  
Источник: sklearn documentation

## RFE (Recursive Feature Elimination)
Рекурсивное исключение признаков с использованием модели.  
Источник: sklearn.feature_selection.RFE

## Permutation Importance
Важность признака, измеряемая через перемешивание его значений.  
Источник: sklearn.inspection.permutation_importance
