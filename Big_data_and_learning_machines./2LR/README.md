## 3. Методология

### 3.1 Глобальная интерпретация (Ноутбук 01)
- Native importance: коэффициенты LogisticRegression и встроенные важности RandomForest
- Permutation importance: важность признаков через перемешивание значений
- Partial dependence plots: зависимость предсказаний от отдельных признаков

### 3.2 Локальный анализ ошибок (Ноутбук 02)
- Perturbation анализ: изменение признаков в ошибочных примерах
- Классификация ошибок: False Positive и False Negative
- Сегментный анализ: распределение ошибок по возрастным группам

---

## 4. Результаты

### 4.1 Сравнение важностей признаков (cardio, set_A)

| Признак | LR native | RF native | Permutation LR | Permutation RF |
|---------|-----------|-----------|----------------|----------------|
| thalach | 0.21 | 0.38 | 0.04 | 0.07 |
| oldpeak | 0.16 | 0.32 | 0.03 | 0.05 |
| age | 0.11 | 0.30 | 0.02 | 0.04 |

Вывод: все методы выделяют thalach как наиболее важный признак.

### 4.2 Partial Dependence

| Признак | Trend | Score Delta |
|---------|-------|-------------|
| thalach | increasing | 0.14 |
| oldpeak | decreasing | 0.11 |
| age | decreasing | 0.07 |

Вывод: thalach показывает наибольшее влияние на предсказание.


### 4.3 Perturbation анализ

Для ошибочных примеров наибольшее изменение вероятности дают:
1. thalach (среднее изменение: 0.08)
2. oldpeak (среднее изменение: 0.05)
3. age (среднее изменение: 0.03)

## 6. Глоссарий

| Термин | Определение |
|--------|-------------|
| Global explanation | Объяснение работы модели на всем датасете |
| Local explanation | Объяснение отдельного предсказания |
| Partial dependence | Зависимость предсказания от одного признака |
| Permutation importance | Важность признака через перемешивание значений |
| Perturbation | Анализ влияния изменения признака на предсказание |
| False Positive | Ложное срабатывание (ошибка I рода) |
| False Negative | Ложный пропуск (ошибка II рода) |

## 7. Выводы

1. Глобальная интерпретация подтверждает результаты отбора признаков из ЛР01: thalach является наиболее важным признаком для прогноза сердечно-сосудистого риска
2. Разные методы интерпретации (native, permutation) дают согласованные результаты, что повышает доверие к выводам
3. Partial dependence plots показывают монотонные зависимости: увеличение thalach повышает вероятность риска, увеличение oldpeak и age понижает
4. Локальный анализ ошибок выявил равномерное распределение FP и FN, модель не смещена в сторону одного типа ошибок
5. Сегментный анализ не показал критических различий в качестве между возрастными группами
6. Perturbation анализ подтверждает, что thalach сильнее всего влияет на изменение предсказания в ошибочных случаях
7. Для улучшения модели рекомендуется сбор дополнительных данных или добавление новых признаков
