# Глоссарий ЛР03

## Generalization gap - разница между метриками на train и validation
## Overfitting - переобучение, когда модель хорошо работает на train но плохо на новых данных
## Validation set - отложенная выборка для подбора гиперпараметров
## GridSearchCV - перебор гиперпараметров с кросс-валидацией
## Validation curve - график зависимости метрики от гиперпараметра
## Baseline - базовая модель с параметрами по умолчанию
