## 3. Методология

### 3.1 Калибровка вероятностей
Сравнение трех вариантов:
- uncalibrated (базовая модель)
- calibrated_sigmoid (Platt scaling)
- calibrated_isotonic (изотоническая регрессия)

Метрики оценки калибровки:
- Brier score: среднеквадратичная ошибка вероятностей
- LogLoss: логарифмическая функция потерь
- ROC AUC: способность разделять классы
- ECE (Expected Calibration Error): ошибка калибровки по бинам

### 3.2 Подбор порога
Перебор значений threshold от 0.10 до 0.95 с шагом 0.05.
Для каждого порога вычисляются:
- precision, recall, f1
- fp_rate, fn_rate
- expected_cost = (FP * 1 + FN * 5) / N

Правило выбора: минимальная expected_cost при ограничении recall >= 0.60.

### 3.3 Финальная проверка
Выбранное правило решения применяется к test один раз.
Дополнительно проводится сегментный анализ ошибок по возрастным группам.

## 6. Глоссарий

| Термин | Определение |
|--------|-------------|
| Калибровка вероятностей | Приведение предсказанных вероятностей к наблюдаемым частотам |
| Brier score | Среднеквадратичная ошибка между вероятностью и фактом |
| ECE | Ожидаемая ошибка калибровки, среднее отклонение по бинам |
| Platt scaling | Калибровка через логистическую регрессию на выходе модели |
| Isotonic calibration | Непараметрическая калибровка изотонической регрессией |
| Threshold | Порог вероятности для принятия решения о классе |
| Expected cost | Математическое ожидание стоимости ошибок |
| Guardrail | Ограничение на минимально допустимое значение метрики |

## 7. Выводы

1. Калибровка вероятностей улучшает качество вероятностных предсказаний: Brier score снизился с 0.241 до 0.228, ECE уменьшился с 0.148 до 0.051

2. Изотоническая калибровка показывает лучшие результаты по сравнению с сигмоидной, но требует больше данных для устойчивости

3. ROC AUC не изменяется при калибровке, так как она сохраняет порядок предсказаний

4. Подбор порога позволил снизить ожидаемую стоимость ошибок по сравнению со стандартным порогом 0.50

5. Ограничение recall >= 0.60 обеспечивает приемлемую полноту модели, гарантируя обнаружение не менее 60% положительных случаев

6. Сегментный анализ не выявил критических различий в качестве работы модели по возрастным группам

7. На синтетических данных абсолютные значения метрик близки к случайным, однако процедура калибровки и выбора порога демонстрирует корректный подход

8. Выбранное правило решения (threshold с min cost при recall guardrail) готово к применению в условиях, где цена FN в 5 раз выше цены FP
