# Глоссарий ЛР05

## Drift (дрейф) - изменение распределения данных со временем
## Covariate drift - сдвиг в распределении признаков
## Prior drift - сдвиг в распределении целевой переменной
## PSI (Population Stability Index) - индекс стабильности популяции
## KS test - критерий Колмогорова-Смирнова для сравнения распределений
## Chi-square test - критерий хи-квадрат для категориальных признаков
## p-value - вероятность получить наблюдаемый результат при верной H0
## Retrain policy - правило принятия решения о переобучении
## Observe - решение наблюдать дальше без переобучения
## Reference window - эталонное окно данных для сравнения
