## Кратко о результатах

### Этап 1
- **Метод** — конкретный способ действия (инструмент)
- **Методология** — общий план, система принципов (стратегия)
- Методология включает в себя методы и определяет, когда их применять

### Этап 2
- Построена схема с характеристиками (цель, уровень, примеры, область применения)
- Приведено 5 примеров из реальной ИТ-практики

### Этап 3
- Разработано приложение для заметок
- Использована методология **Kanban**
- Применены методы: CRUD, MVC, ручное тестирование

## Запуск проекта

```bash
it clone https://github.com/KOTorCAT/6th-semester-/tree/main/Educational_and_research_workshop/1LR/3stage

# 2. Активировать виртуальное окружение
python3 -m venv venv
source venv/bin/activate

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Выполнить миграции (создать базу данных)
python manage.py migrate

# 5. Запустить сервер
python manage.py runserver 8001
```

Открыть в браузере: **http://127.0.0.1:8001**
