### 3.1 Муниципальная пекарня (гражданская задача 1)

**Постановка:** Пекарня выпекает хлеб и булочки. Определить оптимальный план производства для максимизации прибыли.

**Математическая модель:**
```
max Z = 50X + 30Y
2X + Y <= 100  (мука)
X + 2Y <= 80   (время)
Y <= 40        (спрос)
X >= 0, Y >= 0
```

**Ход решения:**
1. Построена допустимая область как пересечение трех полуплоскостей
2. Определены вершины: (0,0), (0,40), (20,40), (40,20), (50,0)
3. Вычислена прибыль в каждой вершине
4. Решение проверено через linprog

**Результат:**
- Оптимальный план: X = 40 хлебов, Y = 20 булочек
- Максимальная прибыль: 2600 рублей
- Ресурсы использованы: мука полностью, время полностью, ограничение по спросу не активно

**Файл:** student/lab_01_student_civil_01.ipynb

### 3.2 Городской ремонтный участок (гражданская задача 2)

**Постановка:** Распределить бригады X и Y для ремонта дорог и тротуаров с минимальными затратами.

**Математическая модель:**
```
min Z = 15X + 12Y
3X + Y >= 18  (дороги)
X + 2Y >= 12  (тротуары)
X >= 0, Y >= 0
```

**Ход решения:**
1. Построена допустимая область (ограничения снизу)
2. Найдены точки пересечения ограничений
3. Задача решена через linprog с преобразованием >= в <=

**Результат:**
- Оптимальный план: X = 4 бригады, Y = 6 бригад
- Минимальные затраты: 132 тыс. рублей
- Оба ограничения активны (дороги и тротуары точно по плану)

**Файл:** student/lab_01_student_civil_02.ipynb

### 3.3 Комплектование полевых рационов (военная задача 1)

**Постановка:** Составить рацион из продуктов A и B с минимальной стоимостью при заданных нормах питательных веществ.

**Математическая модель:**
```
min Z = 10A + 8B
4A + 2B >= 20  (белки)
A + 3B >= 15   (жиры)
2A + 5B >= 30  (углеводы)
A >= 0, B >= 0
```

**Ход решения:**
1. Построена допустимая область (над всеми тремя линиями ограничений)
2. Найдены вершины: (3,4), (2.5,5), (15,0)
3. Вычислена стоимость в каждой вершине

**Результат:**
- Оптимальный рацион: A = 3 единицы, B = 4 единицы
- Минимальная стоимость: 62 рубля
- Белки: 4*3 + 2*4 = 20 (точно по норме)
- Жиры: 3 + 3*4 = 15 (точно по норме)
- Углеводы: 2*3 + 5*4 = 26 (избыток 4 единицы)

**Файл:** student/lab_01_student_military_01.ipynb

### 3.4 Ремонтно-обслуживающий склад (военная задача 2)

**Постановка:** Определить количество запчастей X и Y для хранения с минимальными затратами при ограничении объема склада и минимальной потребности.

**Математическая модель:**
```
min Z = 100X + 80Y
2X + 3Y <= 120  (объем склада)
X >= 20         (потребность X)
Y >= 30         (потребность Y)
```

**Ход решения:**
1. Задача с двусторонними ограничениями (сверху и снизу)
2. Решена через linprog с параметром bounds

**Результат:**
- Оптимальный план: X = 20 единиц, Y = 26.7 единиц
- Минимальные затраты: 4133 рубля
- Склад заполнен полностью: 2*20 + 3*26.7 = 120 м3
- Потребность по X на нижней границе, по Y превышает минимум

**Файл:** student/lab_01_student_military_02.ipynb

## 6. Глоссарий

| Термин | Определение |
|--------|-------------|
| Целевая функция | Линейная функция Z = c1X + c2Y, значение которой оптимизируется |
| Допустимая область | Множество точек, удовлетворяющих всем ограничениям задачи |
| Вершина | Угловая точка допустимой области, кандидат на оптимальное решение |
| linprog | Функция библиотеки scipy для решения задач линейного программирования |
| Симплекс-метод | Алгоритм последовательного перебора вершин для поиска оптимума |
| Активное ограничение | Ограничение, которое в оптимальной точке выполняется как равенство |

## 7. Выводы

1. Линейное программирование является эффективным методом решения оптимизационных задач с линейной целевой функцией и линейными ограничениями

2. Геометрический метод нагляден для задач с двумя переменными, но для большего числа переменных необходимо использовать численные методы

3. Оптимальное решение всегда достигается в одной из вершин допустимой области (или на ребре, если целевая функция параллельна ограничению)

4. Функция linprog автоматизирует поиск решения и работает с задачами любой размерности

5. При использовании linprog важно помнить, что функция решает задачу минимизации. Для максимизации необходимо изменить знаки коэффициентов целевой функции

6. Все четыре задачи успешно решены двумя методами (геометрическим и численным), результаты совпадают

7. Полученные навыки применимы к широкому классу прикладных задач: планирование производства, составление смесей, распределение ресурсов, логистика
