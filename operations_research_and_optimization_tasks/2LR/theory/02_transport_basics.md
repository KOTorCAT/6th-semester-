# Транспортная задача

## Постановка
Есть m поставщиков с запасами a_i и n потребителей со спросом b_j.
Стоимость перевозки единицы от i к j равна c_ij.
Найти план перевозок x_ij минимальной стоимости.

## Баланс
- Закрытая задача: sum(a_i) = sum(b_j)
- Открытая задача: добавляем фиктивного поставщика или потребителя

## Математическая модель
min Z = sum(c_ij * x_ij)

Ограничения:
- sum_j(x_ij) = a_i для всех поставщиков
- sum_i(x_ij) = b_j для всех потребителей
- x_ij >= 0

## Решение через linprog
- Переменные: flatten матрицы X
- A_eq: матрица ограничений-равенств
- b_eq: запасы и спрос
- bounds: (0, None) для всех переменных
