### 3.1 Муниципальное здравоохранение (гражданская задача 1)

**Постановка:** Мэрия распределяет бюджет между поликлиниками (X) и стационарами (Y). Максимизировать охват пациентов при ограничениях по врачам, оборудованию и бюджету.

**Прямая модель (Primal):**
```
max Z = 200X + 300Y

Ограничения:
2X + 3Y <= 120  (врачи)
4X + Y <= 100   (оборудование)
X + 2Y <= 80    (бюджет)
X, Y >= 0
```

**Решение прямой задачи:**
- Поликлиники X = 15.0
- Стационары Y = 30.0
- Охват пациентов Z = 12000

**Анализ запасов ресурсов:**
- Врачи: запас = 0 (дефицит, активное ограничение)
- Оборудование: запас = 10 (избыток, неактивное ограничение)
- Бюджет: запас = 0 (дефицит, активное ограничение)

**Двойственная модель (Dual):**
```
min W = 120y1 + 100y2 + 80y3

Ограничения:
2y1 + 4y2 + y3 >= 200  (поликлиники)
3y1 + y2 + 2y3 >= 300  (стационары)
y1, y2, y3 >= 0
```

**Теневые цены ресурсов:**
- y1 (врачи) = 40.0 ед охвата на одного врача
- y2 (оборудование) = 0.0 (избыток, нулевая ценность)
- y3 (бюджет) = 120.0 ед охвата на единицу бюджета

**Проверка прогноза:**
- Прогноз: +1 врач увеличит охват на 40.0
- Повторное решение с врачами = 121
- Реальное изменение: 40.0
- Прогноз подтвержден

**Проверка сильной двойственности:**
- Z (primal) = 12000
- W (dual) = 12000
- Равенство выполняется

**Файл:** student/lab_03_student_civil_01.ipynb

### 3.2 Социальная защита в зимний период (гражданская задача 2)

**Постановка:** Распределение средств на отопление (X) и питание (Y). Максимизировать полезность при ограничениях по фонду, волонтерам и минимуму питания.

**Прямая модель:**
```
max Z = 40X + 60Y

Ограничения:
5X + 8Y <= 400  (фонд)
2X + Y <= 100   (волонтеры)
Y >= 20         (минимум питания)
X, Y >= 0
```

**Решение прямой задачи:**
- Отопление X = 20.0
- Питание Y = 37.5
- Полезность Z = 3050.0

**Анализ запасов:**
- Фонд: запас = 0 (дефицит)
- Волонтеры: запас = 22.5 (избыток)
- Минимум питания: запас = 17.5 (избыток, Y > 20)

**Теневые цены:**
- y1 (фонд) = 5.0 ед полезности на единицу фонда
- y2 (волонтеры) = 0.0 (избыток)
- y3 (мин питание) = 0.0 (не активно)

**Сравнительный анализ:**
- +1 волонтер: 0 ед полезности (ресурс в избытке)
- +10 единиц фонда: 10 * 5.0 = 50 ед полезности
- Вывод: увеличивать фонд выгоднее, чем привлекать волонтеров

**Экономический смысл:**
Фонд является узким местом. Дополнительные волонтеры бесполезны без увеличения фонда, так как не хватает средств на материалы для отопления и питания.

**Файл:** student/lab_03_student_civil_02.ipynb

### 3.3 Бюджет логистической готовности (военная задача 1)

**Постановка:** Распределение бюджета между топливом (X) и запчастями (Y). Максимизировать боеготовность при ограничениях общего бюджета, минимальных долей и пропорции.

**Прямая модель:**
```
max Z = 50X + 80Y

Ограничения:
X + Y <= 10    (общий бюджет)
X >= 3         (мин топливо)
Y >= 2         (мин запчасти)
X <= 2Y        (пропорция)
X, Y >= 0
```

**Приведение к стандартному виду (все <=):**
```
X + Y <= 10
-X <= -3
-Y <= -2
X - 2Y <= 0
```

**Решение прямой задачи:**
- Топливо X = 5.0 млн руб
- Запчасти Y = 5.0 млн руб
- Боеготовность Z = 650.0 ед

**Анализ запасов:**
- Общий бюджет: запас = 0 (дефицит, активное)
- Мин топливо: запас = 2.0 (X > 3, не активно)
- Мин запчасти: запас = 3.0 (Y > 2, не активно)
- Пропорция: запас = 0 (дефицит, активное)

**Теневые цены:**
- y1 (бюджет) = 65.0 ед боеготовности на 1 млн
- y2 (мин топливо) = 0.0
- y3 (мин запчасти) = 0.0
- y4 (пропорция) = 15.0 ед за изменение пропорции

**Выводы:**
- Общий бюджет — самый ценный ресурс
- Минимальные ограничения не активны (план превышает минимумы)
- Пропорция X = 2Y активна и ограничивает перераспределение
- Дополнительный миллион бюджета даст +65 ед боеготовности

**Файл:** student/lab_03_student_military_01.ipynb

### 3.4 Модернизация ремонтной базы (военная задача 2)

**Постановка:** Распределение средств между ремонтом техники (X) и оборудования (Y). Максимизировать боеготовность при ограничениях по механикам, запчастям и времени.

**Прямая модель:**
```
max Z = 100X + 150Y

Ограничения:
3X + 5Y <= 150  (механики)
4X + 2Y <= 120  (запчасти)
X + 3Y <= 90    (время)
X, Y >= 0
```

**Решение прямой задачи:**
- Ремонт техники X = 16.7 ед
- Ремонт оборудования Y = 20.0 ед
- Боеготовность Z = 4666.7 ед

**Анализ запасов:**
- Механики: запас = 0 (дефицит)
- Запчасти: запас = 13.3 (избыток)
- Время: запас = 23.3 (избыток)

**Теневые цены:**
- y1 (механики) = 33.3 ед боеготовности на механика
- y2 (запчасти) = 0.0 (избыток)
- y3 (время) = 0.0 (избыток)

**Проверка прогнозов:**

Прогноз 1: +5 механиков
- Ожидаемый прирост: 5 * 33.3 = 166.7 ед
- Решение с механиками = 155
- Реальный прирост: 166.7 ед
- Прогноз подтвержден

Прогноз 2: -10 запчастей
- Ожидаемые потери: 10 * 0 = 0 ед (ресурс в избытке)
- Решение с запчастями = 110
- Реальные потери: 0 ед
- Прогноз подтвержден

**Выводы:**
- Механики — единственный дефицитный ресурс
- Запчасти и время в избытке, их изменение не влияет на оптимум
- Увеличение штата механиков прямо пропорционально увеличивает боеготовность
- Сокращение поставок запчастей до определенного предела безопасно

## 7. Глоссарий

| Термин | Определение |
|--------|-------------|
| Прямая задача (Primal) | Исходная задача оптимизации с заданными ресурсами |
| Двойственная задача (Dual) | Задача, назначающая внутренние цены ограничениям primal |
| Теневая цена (Shadow price) | Изменение оптимума при увеличении ресурса на единицу |
| Активное ограничение | Ограничение, выполняющееся как равенство в оптимуме |
| Запас ресурса (Slack) | Остаток ресурса: b_i - (A*x)_i |
| Сильная двойственность | Равенство Z_primal = W_dual в оптимальной точке |
| Анализ чувствительности | Исследование влияния изменения параметров на решение |
| Маргинальная ценность | Предельная полезность дополнительной единицы ресурса |
| Дефицитный ресурс | Ресурс с нулевым запасом и положительной теневой ценой |
| Избыточный ресурс | Ресурс с положительным запасом и нулевой теневой ценой |

## 8. Выводы

1. Двойственная задача предоставляет альтернативный взгляд на проблему оптимизации: вместо выбора действий назначает внутренние цены ограничениям

2. Теневые цены показывают реальную экономическую ценность ресурсов: сколько дополнительных единиц целевой функции можно получить за единицу ресурса

3. Активные ограничения (ресурсы с нулевым запасом) имеют положительные теневые цены и являются узкими местами системы

4. Неактивные ограничения (ресурсы с избытком) имеют нулевые теневые цены: их дополнительное количество бесполезно

5. Сильная двойственность подтверждена во всех задачах: значения целевых функций primal и dual совпадают

6. Прогнозы на основе теневых цен работают точно для малых изменений параметров в пределах устойчивости решения

7. Анализ чувствительности позволяет принимать обоснованные управленческие решения: куда направить дополнительные ресурсы, где можно сократить без потерь

8. В задаче соцзащиты показательно, что дополнительные волонтеры бесполезны без увеличения фонда — ресурсы должны быть сбалансированы

9. В задаче рембазы продемонстрировано, что сокращение избыточного ресурса (запчастей) не влияет на оптимум до достижения границы активного ограничения

10. Методология двойственности и анализа чувствительности применима к широкому кругу задач: распределение бюджета, планирование производства, логистика, военное снабжение
