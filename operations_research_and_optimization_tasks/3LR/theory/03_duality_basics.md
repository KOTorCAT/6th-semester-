# Двойственность и анализ чувствительности

## Прямая задача (Primal)
Выбирает действия при заданных ресурсах.
Пример: max прибыль при ограничениях по материалам.

## Двойственная задача (Dual)
Смотрит на ограничения как в зеркале.
Назначает внутренние цены ресурсам.
Пример: min стоимость ресурсов, обеспечивающая заданную прибыль.

## Ключевые понятия
- Теневая цена (shadow price) - насколько изменится оптимум при увеличении ресурса на 1
- Активное ограничение - ресурс использован полностью (запас = 0)
- Неактивное ограничение - ресурс в избытке (запас > 0, теневая цена = 0)
- Запас ресурса (slack) - b_i - (A*x)_i

## Правила перехода Primal -> Dual
1. Max -> Min, Min -> Max
2. Коэффициенты c становятся правыми частями
3. Правые части b становятся коэффициентами
4. Матрица A транспонируется
5. Знаки неравенств: >= для max, <= для min (при x >= 0)
