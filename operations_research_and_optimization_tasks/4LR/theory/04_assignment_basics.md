# Задача о назначениях

## Постановка
Есть n исполнителей и n задач.
Каждый получает ровно одну задачу.
Назначение (i, j) имеет стоимость c_ij.
Найти назначение минимальной суммарной стоимости.

## Математическая модель
min Z = sum(c_ij * x_ij)

Ограничения:
sum_j(x_ij) = 1 для всех i (каждый исполнитель — одна задача)
sum_i(x_ij) = 1 для всех j (каждая задача — один исполнитель)
x_ij in {0, 1}

## Методы решения
1. Венгерский алгоритм: scipy.optimize.linear_sum_assignment
2. LP-решение: linprog с ограничениями равенства
3. Полный перебор: n! вариантов

## Проверки (assert)
- Каждый исполнитель назначен: sum по строке = 1
- Каждая задача назначена: sum по столбцу = 1
- Размерность: матрица n x n
